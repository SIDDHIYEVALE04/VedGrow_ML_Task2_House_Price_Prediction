House Price Prediction

Project Overview

This project predicts house prices using machine learning regression models.

Dataset

California Housing Dataset from Scikit-Learn.

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

Data Preprocessing

* Missing Value Analysis
* Correlation Analysis
* Outlier Detection
* Feature Selection

Models Used

* Linear Regression
* Ridge Regression
* Random Forest Regressor

Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

Feature Importance

Feature importance was analyzed using Random Forest Regression.
